🏴‍☠️ Welcome to Pirate Llama

This wizard sets up your storage drives, cryptographic identity, and brain model. The choices you make here are permanent — especially your login order.

⚠️ Admin rights required — pirate_llama.exe runs as Administrator. File associations for .sfs and .sfsp are set during this wizard.
🔑 Your username + password (in the order you choose) are hashed into a cryptographic key that gets embedded into every model you convert. Always log in using the same order.
💾 Drive Selection

Select your HDD for cold SFS storage, and your NVMe drive(s) for active model spinup.

⚠️ Store SFS files on HDD. Load to NVMe only when running. Large models on NVMe wear out the drive and waste expensive space.
