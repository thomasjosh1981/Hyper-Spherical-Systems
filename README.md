## 🚀 Quickstart Guide

### 1. Extract the Application
1. Download **`HyperSpherical_v2.0_Beta.zip`**.
2. Right-click and **Extract All** to a folder on your Windows PC (e.g. `C:\HyperSpherical`).

### 2. Launch the Web Application & Live M2M Dashboard
Run `launch_ui.py` using Python:
```cmd
python launch_ui.py
```
This launches the backend server and automatically opens your browser to the interactive **M2M Machine-to-Machine Dashboard & Pirate GUI**.
